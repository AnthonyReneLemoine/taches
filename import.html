Import des données Google Sheets → Firestore

Outil à usage unique — à supprimer après migration

1
Vérifiez la configuration Firebase

Même config que dans index.html. Modifiez les valeurs directement dans ce fichier.

2
Exportez vos données depuis Google Apps Script

Dans votre projet GAS, ajoutez et exécutez cette fonction :

function exporterJSON() {
  const taches = getTaches();
  Logger.log(JSON.stringify(taches, null, 2));
}

Copiez le résultat des logs (Ctrl+Entrée pour exécuter, puis Journaux d'exécution).

3
Collez le JSON exporté
4
Lancez l'import
En attente...
Après la migration : supprimez ce fichier import.html de votre dépôt. Il n'a pas d'authentification et permettrait à n'importe qui d'écraser vos données.
